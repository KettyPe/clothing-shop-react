# Интернет-магазин одежды
 
Тестовое задание на позицию frontend-разработчика. Мобильное приложение интернет-магазина одежды, ориентированное на использование в Telegram.
 
---
 
**Живой деплой:** https://noxer-test.vercel.app/
 
---
 
## О проекте
 
Интернет-магазин с акцентом на мобильные устройства.  
Проект собран с нуля: архитектура, компоненты, хуки, стейт-менеджмент.
 
### Что реализовано
 
- Декомпозиция на переиспользуемые компоненты — Header, Footer, ProductCard, Categories, MainblockBanner, Products, Search
- Бесконечная подгрузка товаров через кастомный хук `useInfiniteScroll` на базе IntersectionObserver API
- Закрытие модальных окон по клику вне области через кастомный хук `useClickOutside`
- Загрузка данных через API и локальный файл данных
- Глобальный стейт через Jotai
- Адаптивная вёрстка с упором на мобильные устройства
 
---
 
## Стек
 
- React, TypeScript
- Vite
- Jotai — стейт-менеджмент
- ESLint, Prettier
 
---
 
## Кастомные хуки
 
`useInfiniteScroll` — бесконечный скролл через IntersectionObserver  
`useClickOutside` — закрытие элемента по клику вне его области
